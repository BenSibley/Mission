-------------------------------------------------------
    Features
-------------------------------------------------------

Ct_theme_name includes the following features:

    * Custom logo
    * Social media icons
    * Show full posts or excerpts
    * Custom excerpt length
    * Custom Read More button text
    * Make first post on blog extra wide
    * Hide/Show display controls
    * Custom CSS

Ct_theme_name also has a premium plugin upgrade called Ct_theme_name Pro that includes the following features:

    * Custom colors
    * Featured sliders
    * Featured videos
    * Additional layouts
    * Flexible header image
    * Google fonts
    * More widget areas
    * Display controls
    * Custom footer text

Ct_theme_name Pro: https://www.competethemes.com/ct_theme_name-pro/

-------------------------------------------------------
    Additional Info:
-------------------------------------------------------

    * All menus in Ct_theme_name can have 1-3 tiers
    * Featured Images have a 2x1 ratio at all screen sizes
    * Ct_theme_name is child theme compatible. Download a starter child theme here:
        https://www.competethemes.com/help/child-theme-ct_theme_name/
